视图共存问题

Freemarker + thymeleaf 同时存在先加载Freemarker 两个优先级一样 视图都能看见
Freemarker + jsp 同时存在先加载Freemarker 优先级前后高于后者， jsp是最低的 视图都能看见
thymeleaf + jsp 同时存在 jsp视图看不见 因为thymeleaf不具备检查视图的能力
Freemarker + thymeleaf + jsp 同时存在 要配置两个检查能力 三个视图就能共存

@JsonFormat(pattern = "yyyy-mm-dd HH:mm:ss， timezone = "Asia/Shanghai") 时间格式化
@JsonPropertyOrder 类似于JsonProperty中的index
