npm install 报错

2023-01-11 17:56
node 版本与git下载下来的版本不一致，需要降低版本来操作（大部分情况下）

1.首先安装node版本管理模块 sudo cnpm install n -g
2.版本降级/升级 sudo n 版本号 例如：sudo n 14.17.4。
3.最后检查自己node版本号 node-v // 修改成功
执行 npm install 命令时，默认使用的是国外的下载源 ，可以通过如下代码配置为使用淘宝的镜像：
npm config set registry https://registry.npm.taobao.org
